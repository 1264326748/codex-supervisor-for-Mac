# Code of Conduct

## 我们的承诺

为了让协作环境对所有人都友好，项目维护者和参与者承诺：

- 以尊重和建设性的方式沟通；
- 聚焦事实与问题本身，不做人身攻击；
- 欢迎不同背景、经验和观点的贡献者。

## 期望行为

- 在 Issue / PR 讨论中保持礼貌、清晰和专业；
- 对反馈给出可复现信息（步骤、日志、环境）；
- 尊重评审意见，围绕代码与方案讨论。

## 不可接受行为

- 侮辱、歧视、骚扰或恶意挑衅；
- 泄露他人隐私或敏感信息；
- 反复发布无关或干扰性内容；
- 任何形式的人身攻击与威胁。

## 执行与处理

维护者有权对违反本规范的行为采取措施，包括但不限于：

- 警告；
- 删除评论；
- 关闭相关讨论；
- 暂停或永久限制参与权限。

## 联系方式

若你在项目社区中遇到不当行为，请通过仓库 Issue 联系维护者，并附上必要上下文，维护者会尽快处理。
