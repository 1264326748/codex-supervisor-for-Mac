# v0.1.0 - Initial open-source release

## Highlights

- 本地桌面控制台：主管窗口 + 多执行窗口并行协同
- 结构化任务下发：支持 `dispatch_json` / `dispatch_batch_json` / `dispatch_all_json`
- 统一待处理队列：集中处理确认提示，支持批量操作
- 实时输出面板：可直接对 supervisor / worker 注入指令
- 会话与日志落盘：支持回放、重连与排查

## Stability

- 主管继续建议默认自动续跑，减少阻塞
- 遗留继续建议可自动解阻塞
- 增加手动发送后阻塞提示与超时提示
- 增加示例占位下发拦截，避免误派发

## Developer Notes

- 项目基于 Electron + React + Vite
- 建议优先使用 tmux 运行时，子进程模式作为回退
- 测试入口：`pnpm test`
- 构建入口：`pnpm build`
